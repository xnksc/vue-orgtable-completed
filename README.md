# SPA Таблица организаций

## Project Setup

```sh
npm install
```

### Compile and Minify for Production

```sh
npm run build
```

### Добавленные функции:
- Сортировка по убыванию|возрастанию
- Фильтрация по имени
- Добавление организации
- Удаление организации
- Пагинация + роутинг
- Небольшая адаптивность
